# Week 3 — Challenges

Two open-ended problems, ~90 min each. **No single right answer.** Unlike the exercises, these ask for judgment, a written recommendation, and numbers that support it — the shape of real growth-analyst work.

1. **[Challenge 1 — Find the aha moment empirically](challenge-01-find-the-aha-empirically.md)** — run the full lift analysis and defend your pick against reach and causality objections.
2. **[Challenge 2 — Redesign onboarding for activation lift](challenge-02-redesign-onboarding-for-lift.md)** — propose a fix for the worst funnel step and quantify its projected impact.

## Before you start

- All three lectures and all three exercises are done.
- You have a working `flags`-style CTE (Lecture 2 §2) and the step-conversion query (Lecture 3 §3) ready to reuse — both challenges build directly on them.

## How these are graded

There's no answer key. Each challenge ships a rubric table instead — use it to self-assess, or trade write-ups with a study partner and grade each other's.
